# Bug Report - BUG-002

## Title

Cart item count does not update after removing a product.

## Project

SauceDemo E-commerce Website

## Severity

Medium

## Priority

Medium

## Description

The cart icon count remains unchanged after a product is removed from the shopping cart.

## Steps to Reproduce

1. Log in with valid credentials.
2. Add a product to the cart.
3. Open the shopping cart.
4. Remove the product.
5. Return to the product page.

## Expected Result

The cart count should update and display zero items.

## Actual Result

The cart count continues displaying the previous item quantity after the product has been removed.
