# Bug Report - BUG-001

## Title

Checkout page allows users to continue without completing required fields.

## Project

SauceDemo E-commerce Website

## Environment

Browser: Google Chrome

Operating System: macOS

## Severity

Medium

## Priority

High

## Description

The checkout information page should prevent users from continuing when required fields are empty. Validation should appear before moving to the next step.

## Steps to Reproduce

1. Log in with valid credentials.
2. Add a product to the shopping cart.
3. Click Checkout.
4. Leave First Name, Last Name, and Zip Code fields empty.
5. Click Continue.

## Expected Result

The system should display validation messages indicating that required fields must be completed.

## Actual Result

The system displays a validation error message and prevents the user from continuing.

## Status

Closed
